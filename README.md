# a-max
Returns maximum value in array

# Install
```sh
$ npm install --save a-max
```

## Examples

```js
var getMax = require('a-max');

console.log(getMax([1, 2, 3, 4]));
// -> 4

```

## Usage

#### `require('a-max')( array )`

**Arguments**:
- `array`: array of Numbers

**Returns**: Returns maximum value from array
